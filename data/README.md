# Data Card — ThisAbled AI

본 디렉터리는 모델 학습·평가에 사용되는 데이터셋을 보관합니다.
**모든 데이터 파일은 `.gitignore`에 의해 commit되지 않습니다** (라이선스·개인정보 보호).

---

## 📦 데이터 출처

| 출처 | 역할 | 라이선스 | 비고 |
|---|---|---|---|
| **Smilegate Unsmile** | 시드 (악성 댓글 라벨) | 출처 라이선스 확인 필요 | https://github.com/smilegate-ai/korean_unsmile_dataset |
| **KOLD** (Korean Offensive Language Dataset) | 시드 | 연구용 (출처 확인 필요) | https://github.com/boychaboy/KOLD |
| **GPT-4o 합성 데이터** | 메인 학습 데이터 | OpenAI 출력물 정책 준수 | 본 프로젝트에서 직접 생성 |

> ⚠️ **라이선스 주의**
> 외부 시드 데이터는 각 출처의 라이선스/이용약관을 반드시 준수합니다.
> 재배포·상업적 이용 시 별도 확인이 필요합니다.

---

## 🏷 라벨 정의 — 모듈 ① 4단계 위험도

| 라벨 | 코드 | 정의 |
|---|---|---|
| 정상 | `0` | 위험 요소 없음. 평범한 일상 대화. |
| 주의 | `1` | 부적절·무례한 표현. 차별/공격성은 미약. |
| 경고 | `2` | 명시적 차별·혐오·괴롭힘. 자기검열 권고 수준. |
| 긴급 | `3` | 자해·위협·심각한 폭력 암시 등 즉시 개입 필요. |

> 상세한 어노테이션 가이드라인은 추후 `docs/annotation_guidelines.md`로 분리 예정.
